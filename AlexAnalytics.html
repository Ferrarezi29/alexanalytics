Conectar ao banco de dados
Para que todos da empresa vejam os mesmos dados, o Alex Analytics usa o Firebase (gratuito) como banco central. Siga os passos abaixo — feito uma vez, nunca mais precisa repetir.
1
Criar conta gratuita no Firebase
Acesse console.firebase.google.com → Clique em "Adicionar projeto" → dê o nome alex-analytics → conclua sem o Google Analytics.
2
Criar o Realtime Database
No menu lateral, clique em Build → Realtime Database → Criar banco → escolha a região us-central1 → inicie no modo teste.

Em seguida, clique na aba Regras e substitua tudo pelo código abaixo, depois clique em Publicar:

{ "rules": { ".read": true, ".write": true } }
✅ Feito isso o banco nunca expira e fica gratuito para sempre.
3
Pegar as credenciais
Na engrenagem ⚙️ → Configurações do projeto → role até "Seus aplicativos" → clique em </> Web → registre o app → copie o objeto firebaseConfig e cole nos campos abaixo.
